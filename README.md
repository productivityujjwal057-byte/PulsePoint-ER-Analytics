# PulsePoint-ER-Analytics
AI-powered Emergency Room Analytics Dashboard built using Python, Streamlit, Flask, and PostgreSQL to analyze ER wait times, peak traffic, ambulance response, triage monitoring, and patient surge forecasting with interactive visualizations and REST APIs.



# Emergency Room Analytics Dashboard

This project is developed for internship purposes.

## Modules Completed
- Data Cleaning & Processing
- ER Wait Time Analytics
- Peak Traffic Hour Analysis
- Critical Patient Trend Analytics
- Ambulance Response Analytics
- Triage Monitoring System
- Patient Surge Forecasting

## Technologies Used
- Python
- Pandas
- Matplotlib
- Streamlit

## Dataset
200+ Emergency Room patient records used for analytics.
